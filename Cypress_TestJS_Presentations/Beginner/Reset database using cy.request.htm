Using cy.task

You can execute Node code during browser tests by calling cy.task

// cypress/plugins/index.jsmodule.exports = (on, config) => {  on('task', {    hello(name) {      console.log('Hello', name)      return null // or Promise    }  })}// cypress/integration/spec.jscy.task('hello', 'World')
